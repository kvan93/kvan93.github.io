About me

I'm a Web/Mobile app developer living in Aarschot and I'm currently in training as a hybrid app developer until April 2021. In this course we will learn the basic front-end technologies such as Bootstrap, Html, Css, Sass and javascript. Later we will learn to use Angular, Ionic framework and connecting to Firebase. I'm looking for a job where I can develop my experience these technologies.

I have prior experience in building complex websites and applications in a .NET environment. I have 3 years professional experience working with C#, VB.NET, SQL Server and basic front-end stack (HTML, CSS, JavaScript/Jquery).

In a part of my free time, I develop new skills or learn new technologies such as Mobile development, Machine Learning and AI, fiddle around on my Raspberry Pi, ...

My hobbies are theatre, futsal (goalkeeper), cycling and casual gaming.

CS50X - Final Project
Android Native app

Tech

  • Android Studio / Java
  • Room Db & SQLite
  • API & Volley framework for HTTP requests
  • Github

Description

This native android application was the final project of an online course given by Harvard. This course was the infamous Cs50 Introduction to computer science where all the basics of computer science were covered. In this course I've learned the fundamental principles of Computer science and programming languages like C, python, SQL and Java for Native Android apps.

The app itself is a simple news website specialized for esports news. The data is collected by targeting the SteamNews API and is stored in a SQLlite Database. In the app, you can browse the latest news articles for multiple esports and save them in a favourites list. The backend is written in Java for Android studio
